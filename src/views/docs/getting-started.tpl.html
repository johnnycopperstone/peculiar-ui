This library is available as a bower package, so with bower installed, you can download this package along with all it's dependencies. bower install peculiar-ui Then add the scripts and css on your page. If you install via bower, then they will be in the bower_components folder. Once you added all the necessary scripts and css on your page, you then need to add peculiar to your angular app. If you don't have one you can simply add pecular as your app on the DOM, otherwise add it as a dependency to your angular app. angular.module('your-app', ['peculiar']);
